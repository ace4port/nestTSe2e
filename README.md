> ## [Nest](https://github.com/nestjs/nest) framework, TypeScript, Prisma, Docker, PostGRES and e2e testing.

### -
## Installation

```bash
$ npm install
```

## Running the app
Ensure you have a docker running. 
```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# e2e tests
$ npm run test:e2e
```
